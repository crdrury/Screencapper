Goliath Auto Screen Capture

Goliath Auto Screen Capture can be used to record a series of screenshots in rapid succession. The user can select an area of the screen to record, assign a hotkey to end recording, and choose a directory to which the screenshots will be saved.

Getting Started
To launch Goliath, simply double-click either Goliath.jar. To run from the command line, use the following:
	java -jar Goliath.jar

Alternatively, a native Windows EXE is can be downloaded and run directly. Native Mac package is coming soon.

Please note that you will need a Java Runtime Environment installed, which can be downloaded from Oracle at https://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html.
